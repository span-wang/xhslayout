    扩展语法速查 颜色/笔刷/标签/提示/导图

    这里只列工具新增写法，普通 Markdown 语法照常可用。

    行内强调

    适合术语、重点、短句。

    • {{red:重点}}红色强调
    • {{blue:术语}}蓝色术语
    • {{green:提示}}绿色提示
    • {{gold:结论}}金色结论
    • ==荧光笔==通用高亮

    笔刷语法

    行内一句或整段重点都能用。

    • {{brush-glow:重点句}}柔光刷
    • {{brush-mist:补充}}雾面刷
    • {{brush-underline:易错点}}下划刷
    • {{brush-tag:关键词}}题签刷
    • :::brush-tip备考建议
    :::brush-tag 高频考点
这里是一整段重点知识
:::

    块级容器

    适合标签、提示框、复盘块。

    • {{block id=summary}}给下一块绑定定位 ID
    • {{#point-a}}短写法，定位下一块内容
    • :::label 高频考点单行标签
    • :::stamp 必背章节印章
    • :::note 复盘提示框标题写在首行
    • :::块内容结束行，需单独占一行
    :::note 复盘
- 概念先记定义
- 做题再看易错点
:::

    树状导图

    用缩进和列表符表示层级。

    • 首行写 :::mindmap 标题
    • 内部用 - 或 1. 列表，缩进 2 个空格表示下一级
    • 末尾单独写 ::: 结束
    :::mindmap 会计职能
- 基本职能
  - 核算职能
  - 监督职能
- 拓展职能
  - 预测经济前景
:::
